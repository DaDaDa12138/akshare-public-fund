## 汽车销量数据

### 1. car_market_total_cpca

**接口名称**: 乘联会-统计数据-总体市场

**数据源**: 乘联会官网

**接口说明**: 获取狭义乘用车或广义乘用车的产量、批发、零售、出口数据

**输入参数**:
- `symbol` (str): 市场类型
  - `"狭义乘用车"` - 狭义乘用车市场
  - `"广义乘用车"` - 广义乘用车市场
- `indicator` (str): 指标类型
  - `"产量"` - 生产数量
  - `"批发"` - 批发数量
  - `"零售"` - 零售数量
  - `"出口"` - 出口数量

**输出字段**:
- `月份` (str): 月份（如"2023-01"）
- 各年份列（如`2023年`, `2022年`等）: 销量数据（单位：万辆）

**数据示例**:
```python
import akshare as ak
df = ak.car_market_total_cpca(symbol="狭义乘用车", indicator="零售")
print(df)
#     月份    2024年   2023年   2022年
# 0  2024-01  215.9   164.9   213.4
# 1  2024-02  142.5   138.5   122.7
```

### 2. car_market_man_rank_cpca

**接口名称**: 乘联会-厂商销售排名

**数据源**: 乘联会官网

**接口说明**: 获取乘用车厂商的单月或累计销售排名数据

**输入参数**:
- `symbol` (str): 统计周期

- `indicator` (str): 销售类型
  - `"批发"` - 批发销量

**输出字段**:
- `厂商` (str): 厂商名称（如"比亚迪"、"一汽大众"）
- 各年份列（如`2024年`, `2023年`等）: 销量排名或数值（单位：万辆）

**数据示例**:
```python
df = ak.car_market_man_rank_cpca(symbol="单月", indicator="零售")
print(df)
#      厂商       2024年   2023年
# 0   比亚迪      36.8    20.3
# 1   一汽大众    18.5    19.2
```

- `O3` (int): 臭氧浓度
- `质量等级` (str): 空气质量等级

**数据示例**:
```python

print(df)
#    时间          AQI  PM2.5  质量等级
```

## 📝 使用注意事项

### 通用说明
1. **数据时效性**: 不同接口的数据更新频率不同
   - 实时数据（票房、空气质量）：5-15分钟更新
   - 日度数据（销量、排名）：每日更新
   - 周度/月度/年度数据：按周期更新

2. **参数格式**: 严格按照文档要求的格式传参
   - 日期格式：YYYYMMDD 或 YYYY-MM-DD
   - 年月格式：YYYYMM 或 YYYY-MM
   - 年份格式：YYYY

3. **数据缺失**: 部分历史数据可能缺失或不完整，需要做好异常处理

4. **网络超时**: 某些接口数据量较大，可能响应较慢，建议设置超时时间 > 30秒

5. **缓存建议**: 对于变化较慢的数据（排行榜、历史数据），建议使用缓存减少API调用

### 常见错误处理
- **KeyError**: 返回数据字段名可能变化，需要灵活处理
- **TimeoutError**: 网络超时，可重试或增加超时时间
- **Empty DataFrame**: 数据暂无或参数错误，检查参数有效性
